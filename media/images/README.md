# 图片文件说明

请将原型机相关的图片文件放入此目录中。

## 支持的格式
- JPG (.jpg, .jpeg)
- PNG (.png)

## 建议的文件命名
- `prototype_front.jpg` - 原型机正面图
- `prototype_side.jpg` - 原型机侧面图
- `prototype_back.jpg` - 原型机背面图
- `components.jpg` - 主要组件图
- `interface.jpg` - 用户界面图

## 图片要求
- 分辨率：建议至少 800x600 像素
- 文件大小：建议控制在 2MB 以内
- 质量：确保图片清晰，可见设备细节
